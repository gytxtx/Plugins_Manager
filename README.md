## 使用方法
1. 将插件以 DLL 文件形式放置在程序运行目录下。
2. 启动程序后，程序会自动扫描运行目录下的 DLL 文件并加载插件信息。
3. 通过界面中的列表框查看插件的详细信息（名称、简介、版本等）。
4. 可通过“加载选定插件”按钮调用选定插件的功能。

## 功能

 - 可获取插件的名称、简介、版本等信息。
 - 可获取插件图标。
 - 插件以 DLL(动态链接库) 形式存储，方便其他程序调用。

## 使用场景

可用于各类程序的插件管理模块中

## 技术栈
